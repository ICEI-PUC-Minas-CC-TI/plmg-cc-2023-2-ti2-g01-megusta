O que é?

O Me Gusta é uma aplicação web com o intuito de auxiliar seus usuários no desenvolvimento de hábitos alimentares saudáveis com a recomendação inteligente de receitas com base nas preferências do usuário. Ao cadastrar, informe suas restrições, selecione alimentos agradáveis ao paladar, alimentos proibidos e comece uma nova rotina alimentar!

A plataforma é um projeto desenvolvido por alunos do curso de Ciência da Computação da PUC Minas para a disciplina de Trabalho Interdisciplinar II: Back-end, do segundo período. O aplicação efetua conexão com o banco de dados PostgreSQL do Microsoft Azure e conta com recursos de sistemas inteligentes para refinar as recomendações de receitas. Saiba mais sobre a equipe em Sobre Nós.
